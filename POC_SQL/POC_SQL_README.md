# POC_SQL — SQL Aplicado

## Propósito

Estudo de SQL na prática: modelagem relacional, consultas (`SELECT`, `JOIN`, filtros), e operações de manipulação de dados (`INSERT`, `UPDATE`, `DELETE`).

---

## Estrutura

```
POC_SQL/
├── POC_SQL_Relational_Model/
│   └── examples/
├── POC_SQL_Search/
│   ├── examples/
│   └── exercises/
└── POC_SQL_Insert_Update_Delete/
    └── examples/
```

---

## Submódulos

| Submódulo | Foco |
|-----------|------|
| `POC_SQL_Relational_Model` | Modelagem de tabelas, chaves primárias/estrangeiras, normalização |
| `POC_SQL_Search` | `SELECT`, `WHERE`, `JOIN`, `GROUP BY`, `ORDER BY`, subqueries |
| `POC_SQL_Insert_Update_Delete` | DML — inserção, atualização e remoção de registros |

---

## Examples vs Exercises

| | `examples/` | `exercises/` |
|-|-------------|--------------|
| **O quê** | Scripts SQL com queries demonstrativas | Consultas a resolver com base em um schema |
| **Como usar** | Executar e analisar os resultados | Escrever a query antes de ver a solução |

---

## Como estudar

Siga a ordem: Relational Model → Search → Insert/Update/Delete. Complementa diretamente o módulo `POC_JDBC`.
