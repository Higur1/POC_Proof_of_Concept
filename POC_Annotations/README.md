# POC_Annotations — Anotações Customizadas em Java

## Propósito

Estudo de criação e processamento de anotações customizadas em Java, incluindo uso com Reflection e configuração de segurança via annotations.

---

## Estrutura

```
readmes/
├── examples/       # Exemplos de anotações built-in e customizadas
└── exercises/
-e     ├── CustomAnnotationSecurityConfig/
    └── CustomAnnotationWithReflection/
```

---

## Examples vs Exercises

| | `examples/` | `exercises/` |
|-|--------------|----------------|
| **O quê** | Demonstrações comentadas de cada conceito | Problemas com solução completa |
| **Como usar** | Leitura e análise do código | Comparar com sua própria tentativa |
| **Foco** | Como funciona | Para que serve |

---

## Como estudar

1. Leia o código em `examples/` na ordem das pastas.
2. Tente resolver o exercício antes de abrir `exercises/`.
3. Compare sua abordagem com a solução existente.
