# POC_Logging — Logging em Java

## Propósito

Estudo de logging estruturado com Log4j2/SLF4J. Cobre níveis de log, configuração de appenders (console, arquivo) e boas práticas de uso em aplicações Java.

---

## Estrutura

```
readmes/
├── examples/       # Configuração de loggers e appenders
└── exercises/
-e     └── LoggingToDifferentDestinations/
```

---

## Examples vs Exercises

| | `examples/` | `exercises/` |
|-|--------------|----------------|
| **O quê** | Demonstrações comentadas de cada conceito | Problemas com solução completa |
| **Como usar** | Leitura e análise do código | Comparar com sua própria tentativa |
| **Foco** | Como funciona | Para que serve |

---

## Como estudar

1. Leia o código em `examples/` na ordem das pastas.
2. Tente resolver o exercício antes de abrir `exercises/`.
3. Compare sua abordagem com a solução existente.
