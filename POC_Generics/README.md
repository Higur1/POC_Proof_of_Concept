# POC_Generics — Tipos Genéricos em Java

## Propósito

Estudo de generics em Java: type parameters, bounded wildcards, métodos genéricos e classes genéricas. Inclui uso de generics com interfaces (`Closeable`) e implementação de estruturas genéricas como matrizes.

---

## Estrutura

```
readmes/
├── examples/       # Declaração e uso de tipos genéricos
└── exercises/
-e     ├── CloseAllResources/
    └── Generic3x3MatrixImplementation/
```

---

## Examples vs Exercises

| | `examples/` | `exercises/` |
|-|--------------|----------------|
| **O quê** | Demonstrações comentadas de cada conceito | Problemas com solução completa |
| **Como usar** | Leitura e análise do código | Comparar com sua própria tentativa |
| **Foco** | Como funciona | Para que serve |

---

## Como estudar

1. Leia o código em `examples/` na ordem das pastas.
2. Tente resolver o exercício antes de abrir `exercises/`.
3. Compare sua abordagem com a solução existente.
