# POC_InnerClass — Classes Internas em Java

## Propósito

Estudo dos quatro tipos de classes internas em Java: regular (non-static), estática, anônima e local. Cada tipo tem seu caso de uso e escopo de acesso distintos.

---

## Estrutura

```
readmes/
├── examples/       # Um exemplo por tipo de inner class
└── exercises/
-e     ├── ExecutingRunnableTask/
    ├── HeroAndWeaponMechanics/
    └── JVMMemoryStats/
```

---

## Examples vs Exercises

| | `examples/` | `exercises/` |
|-|--------------|----------------|
| **O quê** | Demonstrações comentadas de cada conceito | Problemas com solução completa |
| **Como usar** | Leitura e análise do código | Comparar com sua própria tentativa |
| **Foco** | Como funciona | Para que serve |

---

## Como estudar

1. Leia o código em `examples/` na ordem das pastas.
2. Tente resolver o exercício antes de abrir `exercises/`.
3. Compare sua abordagem com a solução existente.
