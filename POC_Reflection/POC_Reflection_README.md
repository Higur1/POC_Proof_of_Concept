# POC_Reflection — Reflection API em Java

## Propósito

Estudo da Reflection API para inspeção e manipulação de classes, campos e métodos em tempo de execução. Inclui injeção de dependências simplificada e geração de `toString` via reflexão.

---

## Estrutura

```
readmes/
├── examples/       # Inspeção de objetos, campos e métodos
└── exercises/
-e     ├── InspectingGetters/
    └── ObjecToStringUsingReflection/
```

---

## Examples vs Exercises

| | `examples/` | `exercises/` |
|-|--------------|----------------|
| **O quê** | Demonstrações comentadas de cada conceito | Problemas com solução completa |
| **Como usar** | Leitura e análise do código | Comparar com sua própria tentativa |
| **Foco** | Como funciona | Para que serve |

---

## Como estudar

1. Leia o código em `examples/` na ordem das pastas.
2. Tente resolver o exercício antes de abrir `exercises/`.
3. Compare sua abordagem com a solução existente.
